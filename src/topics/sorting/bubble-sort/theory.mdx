# Bubble Sort

Bubble Sort is one of the simplest sorting algorithms. It repeatedly compares adjacent elements and swaps them if they are in the wrong order.

---

## Overview

Bubble Sort repeatedly passes through an array until no swaps are needed.

---

## Prerequisites

- Arrays
- Loops
- Swapping

---

## Theory

During every pass, the largest element "bubbles" toward the end of the array.

---

## Complexity

| Case | Time |
|------|------|
| Best | O(n) |
| Average | O(n²) |
| Worst | O(n²) |

Space Complexity: **O(1)**

---

## Advantages

- Easy to understand
- In-place sorting
- Stable algorithm

---

## Disadvantages

- Very slow for large datasets

---

## Applications

- Teaching sorting concepts
- Very small datasets
